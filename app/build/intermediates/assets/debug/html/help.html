TEXT EXTRACTOR(OCR Text Scanner)

Version 1.0.00

Help

Follow these steps:

1) First, select the area of the text you need to extract.

2) Then, touch the capture button on the screen.

3) Please wait until the extraction process completed.

4) Now you will redirect to the screen which contains the extracted text.

5) In this screen, you can use menu bar to perform "Copy", "Share", "Read", "Clear", "Help", "About" & "Exit" functions.
